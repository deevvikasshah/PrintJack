# PrintJack

A full-stack MERN ecommerce platform for custom printing services.

## Quick Start

1. Install backend dependencies: `cd server && npm install`
2. Install frontend dependencies: `cd client && npm install`
3. Copy `server/.env.example` to `server/.env` and fill in API keys
4. Seed database: `cd server && npm run seed`
5. Start backend: `cd server && npm run dev`
6. Start frontend: `cd client && npm run dev`

See [AGENTS.md](./AGENTS.md) for full documentation.
